webpack

 webpack 是一个模块打包器。它的主要目标是将 JavaScript 文件打包在一起,打包后的文件用于在浏览器中使用,但它也能够胜任转换
npm install @webpack -g
1.打包
webpack src/main.js dist/bulde.js 
2.webpack.config.js  COMMON.JS 模块 entry 导入 output导出
3.局部安装webpack npm install webpack -save-dev(开发时依赖)   -save(生成运行)
  在package.json中 dependencies(生成环境依赖) devDependencies(开发时依赖)
4.css解析 css-loader style-loader 
5.less解析 less-loader less对less进行编译

vue-cli 脚手架

1.使用提前
1.1 node 8.9以上
1.2 webpack
2.脚手架安装 
 脚手架3:npm install @vue/cli -g 基于webpack4 移除static文件夹 新增public文件夹
 脚手架2:npm install @vue/cli-init -g 基于webpack3  
 区别:
 vue-cli3 路由keep-alive引用为作用域插槽
          router-link 取消tag标签
3.node为js提供了运行环境 
4.runtime-compiler 和 runtimeonly区别
  第一:入口文件中需要template->ast->render->虚拟dom->UI
  第二:直接 render->虚拟dom->ui   render:h=>h(app) 实际上是用了createElement创建app标签元素
  性能更高，代码也更少，项目大小也更小
