传统方式布局

标准流
浮动
给行内元素添加 浮动则变为行内块元素
浮动会影响后面的标准流v
1:浮动与父元素标准流搭配 父元素不定义高度
1.1 则会产生后面的标准流顶上去则要用到清楚浮动
闭合浮动
1)隔墙法  一个块级元素定义css clear:both 清除浮动
   缺点:添加无意义的标签
2)overflow 但是溢出的内容会被隐藏
3)after伪元素
   clearfix:after{
      content:"";
      display:block;
      clear:both;
      height:0;
      visibility:hidden;
      }
   双为元素
   clearfix:before,clearfix:after{
      content:"";
      display:table;
   }
   clearfix:after{
   clear:both
   }
照顾低版本加上
.clearfix{
   *zoom:1
   }
3).1 before after 伪元素是行内元素 必须有content 示例:遮挡层,清楚浮动
2:一浮全浮

display 和 visibility 区别

display 不保留原来的位置

外边距塌陷问题

父盒子添加边框，内边距，overFOLW 或者加定位

定位模式

相对定位:
1.相对自己原来的位置移动
2.原来的位置还保留
绝对定位:
1.如果祖元素没有定位,则按照浏览器来定位
2.离最近有祖先元素定位作为参考点
3.不占有原来的位置
居中 : left:50% margin-left 盒子的一半
定位重叠性
只有定位才能设置z-index

定位特性

1.行内元素添加绝对定位或者固定定位则可以设置宽度高度
2.块级元素添加绝对定位或者固定定位则高度宽随着内容的大小

弹性布局

任何一个容器都能用弹性布局
采用flex布局的元素叫做容器,而子元素称为项目

flex:1 分配剩余空间  - ———————————— -  值为占比分
让所有弹性盒模型对象的子元素都有相同的长度，且忽略它们内部的内容：
flex-direction 设置主轴的方向  row-reverse 反方向    column Y轴为主轴
justify-content 使用这个先确定好主轴 flex-end 以尾部对齐但是没有反方向  center  space_around 平分剩余空间  space_between 先两边贴,再平分
flex-wrap wrap 自动换行
align-items 设置侧轴排序  center  flex-start flex-end
align-content 多行 单行没效果   space-between space-around
flex-flow:复合属性 同时设置 主轴的方向(flex-direction) 和 是否换行(flex-wrap)
