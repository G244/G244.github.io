linux ftp安装

 rpm -qa | grep vsftpd  查看是否安装vsftp
如果没有，就安装，并设置开机启动
yum -y install vsftpd
chkconfig vsftpd on

ssh传输文件

yum install ssh
ssh root@ip  连接远程服务器
scp -p 传送文件
ssh root@ip "mkdir -p /redis/data/140/test" 创建文件
ssh-keygen -t rsa 生成密钥
cd/root/.ssh 里面有id_rsa(私钥)  id_rsa.pub(公钥)
发送公钥到远程服务器 ssh-copy-id -i ~/.ssh/id_rsa.pub ip

创建用户

useradd ftpadmin -g ftp -d /opt/admin 
-d 指定目录      
passwd 添加密码
userdel -r 删除用户
 创建者-同个用户组-其他用户
权限: drwxr-xr-x
查看用户 /etc/passwd
更改文件目录拥有者:chwon ftp:root 拥有者:群体使用者
更改文件目录权限:chmod ug+w,o-w file1.txt  拥有者和同一个群体可写入,其他的人不可写入 | a+r 所有人都可读 r=4，w=2，x=1

getsebool命令是用来查询SElinux

SElinux 安全策略 来检测决定是否可以接入到资源
getenforce 查看SElinux状态 
  Enforcing：强制模式
 Permissive：宽容模式 警告
setenforce 0 设置为宽容模式
修改/etc/selinux/config 文件 disabled

ftp

ftp_nlist() 函数返回 FTP 服务器上指定目录的文件列表。    
ftp_pwd() 函数返回指定 FTP 连接的当前目录名称。    
ftp_chdir() 改变 FTP 服务器上的当前目录。     
ftp_pasv() 在被动模式中，数据连接是由客户机来初始化的，而不是服务器。这在客户机位于防火墙之后时比较有用

 ASCII模式和BINARY模式的区别是回车换行的处理，
 binary模式不对数据进行任何处理，
 asci模式将回车换行转换为本机的回车字符

sftp

ssh2_auth_public_key 秘钥登录
ssh2_scp_send  传输文件
ssh2_scp_recv  下载文件
  • ppk 
1.服务器的ppk文件加Authorized_key 中的服务器公钥
2.本地的ppk文件加Authorized_key中的本地公钥
3.本地服务器公钥匹配Authorized_key中的本地公钥
